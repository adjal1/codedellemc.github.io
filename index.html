EMC CODE & TOOLS

  • ViPRuby

    Ruby Library for ViPR Controller Functions

  • ViPR Java SDK

    Java SDK for ViPR Object Data Services.

  • ViPR Python SDK

    Python SDK for ViPR Object Data Services.

  • ViPR .NET SDK

    .NET 3.5 SDK for ViPR Data Services.

  • vagrant-scaleio

    Vagrantfile to create a three VM EMC ScaleIO lab setup

  • ruby + vipr code

    Sample code that shows how to log-in to the ViPR Controller using Ruby. Also demonstrations the ease of the REST API to gather information and parse it to show different types of information.

  • Cinder Driver for ViPR

    The EMC ViPR Cinder driver contains both an ISCSIDriver as well as a FibreChannelDriver, with the ability to create/delete and attach/detach volumes and create/delete snapshots, etc.

  • ViPRSync

    A bulk copy utility that can move data between various systems in parallel

  • puppet-scaleio

    A Puppet module for installing, and configuring ScaleIO data services components.

  • vagrant-puppet-scaleio

    Vagrant files that prepare a Puppet Master and Agent nodes to configure ScaleIO data services using a Puppet module.

  • XtremLib

    A Powershell module that manages XtremIO storage arrays.

  • Fast Report

    A Python script for displaying FAST VP information per Symmetrix device.

DOCUMENTATION/TUTORIALS

  • Deploying Hadoop on EMC Isilon

    Describes how to create a Hadoop environment Cloudera/Pivotal HD and EMC Isilon Scale-Out NAS for HDFS accessible shared storage. VMware Big Data Extensions is used to provision & manage compute resources

  • Configuring ScaleIO with Puppet

    Describes the module parameters and provides working examples using Vagrant for a full deployment of a Puppet Master node, agents, the module, and a ScaleIO cluster.
